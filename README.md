## 표 만들기  

</br>

    |     | 1 번 | 2 번 | 3 번 |
    | --- | --- | --- | --- |
    | 철수 |  O  |  X  |  O  |
    | --- | --- | --- | --- |
    | 영희 |  X  |  X  |  O  |
